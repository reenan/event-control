O desenvolvimento front-end da aplicação será feito utilizando ReactJS.

Para uma melhor experiência, é desejado e recomendado que o desenvolvedor tenha um bom conhecimento de JavaScript e que leia a documentação do ReactJS disponível no link acima.

O ambiente de desenvolvimento front-end conta com algumas outras ferramentas:

  • JavaScript ES6 sendo parseado pelo Babel;
  • Estilo SCSS sem conversão para CSS - Embutido direto no JavaScript;
  • Classe API para requisições;
  • Ambiente com hot-reload;
  • Webpack como gerenciador de tarefas;
  • Roteamento com React Router
Componentes já disponíveis:
Segue a lista de componentes reutilizáveis já criados no projeto:
  • Ícone (icon.jsx) [icon, size]
  • Grid (grid.jsx) {row: [gutter], col: [size]}
  • Avatar (avatar.jsx) [url, size]
  • Text (text.jsx) [text, size]
  • Button (button.jsx) [text, size, onClick]
Mixins do SCSS
Segue a lista de mixins configuradas no SCSS:
  • transition($args...)
  • box-shadow($size, $color)
  • text-overflow($type)
  • transform($transform)
  • aspect-ratio($width, $height)
  • focal-point-x($x)
  • focal-point-y($y)
  • keyframes($name)
  • animation($content)
  • clearfix()
  • box-sizing($value)
  • vertical-align($value)
  • linear-gradient($args...)
